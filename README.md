# Campaign-Data-visualization
An interactive dasboard shows that the analyzed ad campaigning dataset by evaluating performance, engagement and ROI.

 Marketing Campaign Dashboard (Power BI)

 Overview:

This project is a Power BI dashboard built to analyze marketing campaign performance using real-world advertising data.
It provides insights into user engagement, conversions, and return on investment (ROI).


 Objectives

- Monitor campaign performance
- Analyze user behavior by age and gender
- Identify top-performing ads
- Evaluate conversion efficiency


 Dataset

The dataset includes:

- Impressions – Number of times ads were shown
- Clicks – Number of clicks on ads
- Spent – Total campaign cost
- Total_Conversion – Leads generated
- Approved_Conversion – Successful conversions
- Age – User age group
- Gender – User gender
- Ad ID – Unique ad identifier


 Dashboard Features

KPI Metrics

- CTR (Click Through Rate)
- Total Clicks (38K)
- Total Impressions (213M)
- Total Spend (59K)


 Demographic Analysis

- Click distribution by Age Group
- Click distribution by Gender
- Approved conversions segmented by age & gender

 Conversion Analysis

- Comparison of Total vs Approved Conversions
- Helps identify drop-offs in conversion process

 ROI Analysis

- ROI (%) displayed using a gauge chart (83.79%)
- Evaluates campaign profitability


 Ad Performance Table

- Displays impressions by Ad ID
- Helps identify high-performing ads


 Key Insights

- Age group 45–49 shows highest clicks
- Female users generate more clicks than male users
- Large gap between total and approved conversions
- Some ads have high impressions but low conversion rates


 Tools Used

- Power BI
- DAX (for calculated measures)
- Excel (data source)

 Conclusion:

This dashboard provides a clear and interactive way to analyze marketing campaign performance and supports data-driven decision-making.
